## 📖 Описание

Executive Casebook — это не просто резюме. Это профессиональный сайт-визитка, который показывает подход, принципы и образ мышления, лежащие в основе управленческих решений.

Сайт адаптирует рассказ под того, кто его читает: собственник бизнеса увидит архитектора коммерческой системы, а директор по маркетингу — стратега брендов и роста.

---

## 🧩 Особенности

- **3 перспективы**: Бизнес / Маркетинг / Стратегия
- **Два языка**: RU / EN (переключение без перезагрузки)
- **Адаптивный дизайн** в стиле Apple
- **Интерактивные кейсы** с реальными цифрами
- **Сеть-паутина** для цифрового следа
- **Адаптивные компетенции** (меняют смысл под перспективу)

---

## 🛠️ Технологии

- **HTML5, CSS3** — семантическая вёрстка, CSS-переменные
- **JavaScript (ES6+)** — динамический рендеринг, переключение языка/перспектив
- **Node.js + Express** — серверная часть
- **Nodemailer** — отправка писем через SMTP (mail.ru)
- **Google Fonts** — шрифт Inter
